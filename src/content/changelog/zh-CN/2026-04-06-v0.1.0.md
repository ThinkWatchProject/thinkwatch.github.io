ThinkWatch 现已进入公开预览阶段。此次首个标记版本包含双端口架构（网关 `:3000`，控制台 `:3001`）、虚拟 API 密钥生命周期管理、滑动窗口限流、熔断器，以及统一日志检索。

Helm Chart 与 Distroless 容器镜像均已发布，同时支持 Docker Compose 自托管部署。
