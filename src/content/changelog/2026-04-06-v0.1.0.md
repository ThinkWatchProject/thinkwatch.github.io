ThinkWatch is now in public preview. This first tagged release ships the dual-port architecture (gateway `:3000`, console `:3001`), virtual API key lifecycle management, sliding-window rate limiting, circuit breakers, and the unified log explorer.

The Helm chart and distroless container images are available alongside Docker Compose for self-hosted deployments.
